# المرجع السريع لصيغ اللوحة الرئيسية والتقرير السنوي
## نظام مالي ذكي متكامل - طبقة لوحة المعلومات الفنتك

> هذا الملف يجمع كل الصيغ المعتمدة في وثيقة العمارة [`07_dashboard_architecture.md`](./07_dashboard_architecture.md) في كتل قابلة للنسخ المباشر إلى أيّ خلية في Google Sheets. لكل صيغة وصف عربي قصير، والمرجع المرجعي لخلية الكتابة الموصى بها، ومتغيّرات بديلة عند توفّرها.
>
> النطاق المُسمّى الوحيد المُستهلَك من المرحلة الأولى هو `rng_ActiveFormat`. جميع المُجمِّعات الأخرى تقرأ مباشرة من الأوراق الشهريّة الـ 12 ومن ورقة `الأهداف المالية والادخار`.

---

## 1. إجمالي الدخل السنوي - بطاقة الـ KPI رقم 1

**الوصف:** جمع كل خلايا \"الدخل الفعلي\" (`E10:E28`) عبر 12 ورقة شهريّة.

**الخليّة المستهدفة:** `B5` في ورقة `اللوحة الرئيسية والتقرير السنوي`.

```
=SUM('يناير'!E10:E28, 'فبراير'!E10:E28, 'مارس'!E10:E28, 'أبريل'!E10:E28, 'مايو'!E10:E28, 'يونيو'!E10:E28, 'يوليو'!E10:E28, 'أغسطس'!E10:E28, 'سبتمبر'!E10:E28, 'أكتوبر'!E10:E28, 'نوفمبر'!E10:E28, 'ديسمبر'!E10:E28)
```

## 2. إجمالي المصروفات السنوي - بطاقة الـ KPI رقم 2

**الوصف:** جمع كل خلايا \"المصروف الفعلي\" (`E33:E62`) عبر 12 ورقة شهريّة.

**الخليّة المستهدفة:** `F5`.

```
=SUM('يناير'!E33:E62, 'فبراير'!E33:E62, 'مارس'!E33:E62, 'أبريل'!E33:E62, 'مايو'!E33:E62, 'يونيو'!E33:E62, 'يوليو'!E33:E62, 'أغسطس'!E33:E62, 'سبتمبر'!E33:E62, 'أكتوبر'!E33:E62, 'نوفمبر'!E33:E62, 'ديسمبر'!E33:E62)
```

## 3. صافي الربح السنوي - بطاقة الـ KPI رقم 3

**الوصف:** الفرق بين إجمالي الدخل وإجمالي المصروفات.

**الخليّة المستهدفة:** `J5`.

```
=B5 - F5
```

---

## 4. إجمالي الأصول - بطاقة الـ KPI رقم 4

**الوصف:** المبلغ المدّخر في ورقة الأهداف + الفائض السنوي المتراكم من المحرّك.

**الخليّة المستهدفة:** `N5`.

```
='الأهداف المالية والادخار'!D2 + _DashboardEngine!O5
```

---

## 5. إجمالي الالتزامات - بطاقة الـ KPI رقم 5

**الوصف:** \"إجمالي القسط الشهري المطلوب\" من ورقة الأهداف × 12 (تقدير سنوي).

**الخليّة المستهدفة:** `R5`.

```
='الأهداف المالية والادخار'!B3 * 12
```

> صيغة مؤقّتة (Placeholder)؛ تُستبدَل عند إضافة ورقة \"الديون والالتزامات\" المنفصلة لاحقاً.

---

## 6. معدّل الادخار - بطاقة الـ KPI رقم 6

**الوصف:** `(إجمالي الدخل - إجمالي المصروفات) / إجمالي الدخل`. تُنسَّق الخليّة كنسبة مئويّة.

**الخليّة المستهدفة:** `V5`.

```
=IFERROR((B5 - F5) / B5, 0)
```

---

## 7. صيغة خط الاتجاه (Micro-trend) - الصيغة العامّة

**الوصف:** نصّ يبدأ بـ `▲` أو `▼` مع نسبة مئويّة مُوقَّعة. يُعيد `-` عندما لا توجد قيمة سابقة.

```
=IF(prior_value=0, "-",
    IF(current_value >= prior_value,
       "▲ " & TEXT((current_value - prior_value) / prior_value, "0.0%"),
       "▼ " & TEXT((current_value - prior_value) / prior_value, "0.0%")))
```

استبدل `current_value` بمرجع `_DashboardEngine!H2..H7` و`prior_value` بمرجع `_DashboardEngine!I2..I7` بحسب البطاقة.

### 7.1 مثال ملموس - بطاقة الدخل (`B7`)

```
=IF(_DashboardEngine!I2=0, "-",
    IF(_DashboardEngine!H2 >= _DashboardEngine!I2,
       "▲ " & TEXT((_DashboardEngine!H2 - _DashboardEngine!I2) / _DashboardEngine!I2, "0.0%"),
       "▼ " & TEXT((_DashboardEngine!H2 - _DashboardEngine!I2) / _DashboardEngine!I2, "0.0%")))
```

كرّر بإحلال أرقام الصفوف لـ `H` و`I` (3, 4, 5, 6, 7) لكل بطاقة.

## 8. التنسيق الشرطي - لون اتجاه البطاقات

**الوصف:** قواعد `Custom formula is` للنطاق الذي يحوي خلايا الاتجاه (`B7, F7, J7, N7, R7, V7`). تُكتب القاعدة بمرجع الخليّة الأولى من النطاق المحدَّد.

```
=LEFT($B7,1)="▲"           (خلفية #10B981 - نصّ #FFFFFF)
=LEFT($B7,1)="▼"           (خلفية #EF4444 - نصّ #FFFFFF)
```

## 9. شبكة Chart 1 - المقارنة الشهريّة (محرّك العمود B)

**الوصف:** صيغة كل خليّة في عمود `الدخل الفعلي` على ورقة `_DashboardEngine`. الصيغة تتغيّر فقط بتبديل اسم الورقة الشهريّة.

**الخلايا المستهدفة:** `_DashboardEngine!B2..B13`.

```
=SUM('يناير'!E10:E28)        (B2)
=SUM('فبراير'!E10:E28)      (B3)
=SUM('مارس'!E10:E28)         (B4)
=SUM('أبريل'!E10:E28)        (B5)
=SUM('مايو'!E10:E28)         (B6)
=SUM('يونيو'!E10:E28)        (B7)
=SUM('يوليو'!E10:E28)        (B8)
=SUM('أغسطس'!E10:E28)        (B9)
=SUM('سبتمبر'!E10:E28)       (B10)
=SUM('أكتوبر'!E10:E28)       (B11)
=SUM('نوفمبر'!E10:E28)       (B12)
=SUM('ديسمبر'!E10:E28)       (B13)
```

نظيرها في عمود `المصروف الفعلي` (`C2..C13`) باستبدال `E10:E28` بـ `E33:E62`.

عمود `صافي الربح` (`D2..D13`):

```
=B2-C2          (D2 - اسحب حتى D13)
```

## 10. مُجمِّع فئة عبر 12 شهراً - الصيغة الموسَّعة

**الوصف:** يجمع المصروف الفعلي لفئة واحدة (مثلاً `السكن`) عبر كل الأوراق الشهريّة. يُستخدَم في Chart 2 (Waterfall) وفي Chart 4 (Doughnut Expenses).

```
=SUMIF('يناير'!B33:B62, "السكن", 'يناير'!E33:E62)
 + SUMIF('فبراير'!B33:B62, "السكن", 'فبراير'!E33:E62)
 + SUMIF('مارس'!B33:B62, "السكن", 'مارس'!E33:E62)
 + SUMIF('أبريل'!B33:B62, "السكن", 'أبريل'!E33:E62)
 + SUMIF('مايو'!B33:B62, "السكن", 'مايو'!E33:E62)
 + SUMIF('يونيو'!B33:B62, "السكن", 'يونيو'!E33:E62)
 + SUMIF('يوليو'!B33:B62, "السكن", 'يوليو'!E33:E62)
 + SUMIF('أغسطس'!B33:B62, "السكن", 'أغسطس'!E33:E62)
 + SUMIF('سبتمبر'!B33:B62, "السكن", 'سبتمبر'!E33:E62)
 + SUMIF('أكتوبر'!B33:B62, "السكن", 'أكتوبر'!E33:E62)
 + SUMIF('نوفمبر'!B33:B62, "السكن", 'نوفمبر'!E33:E62)
 + SUMIF('ديسمبر'!B33:B62, "السكن", 'ديسمبر'!E33:E62)
```

استبدل `"السكن"` بمرجع الخليّة (مثلاً `L2`) لجعل الصيغة عامّة قابلة للسحب عبر العمود.

نظيرها لكتلة الدخل (Chart 3 - Doughnut Income): استبدل `B33:B62` بـ `B10:B28`، و`E33:E62` بـ `E10:E28`.

## 11. مُجمِّع فئة - البديل المُدمَج عبر `INDIRECT`

**الوصف:** نسخة مُدمَجة تستهلك عمود مساعد على المحرّك (`P2:P13`) يحوي أسماء الأشهر الـ 12. تقلّل طول الصيغة بشكل كبير. عيبها أنّ `INDIRECT` تُجبر إعادة الحساب.

**شرط مسبق:** اكتب أسماء الأشهر العربيّة الـ 12 في `_DashboardEngine!P2:P13`.

```
=SUMPRODUCT(
   ARRAYFORMULA(
     IFERROR(
       SUMIF(INDIRECT("'" & $P$2:$P$13 & "'!B33:B62"), L2, INDIRECT("'" & $P$2:$P$13 & "'!E33:E62")),
       0
     )
   )
)
```

نظيرها لكتلة الدخل: استبدِل `B33:B62` و `E33:E62` بـ `B10:B28` و `E10:E28`. واستبدل `L2` بـ `I2`.

## 12. درجة الصحّة المالية (Health Score Composite)

**الوصف:** صيغة مُركَّبة من 0 إلى 100 تجمع ثلاث مكوّنات بأوزان: نسبة الادخار (40 نقطة) + الانضباط الميزاني (30 نقطة) + تقدّم الأهداف (30 نقطة).

**الخليّة المستهدفة:** `_DashboardEngine!O2`.

```
=ROUND(
   40 * MAX(0, MIN(1, IFERROR(('اللوحة الرئيسية والتقرير السنوي'!B5 - 'اللوحة الرئيسية والتقرير السنوي'!F5) / 'اللوحة الرئيسية والتقرير السنوي'!B5, 0))) +
   30 * MAX(0, MIN(1, IFERROR(1 - ('اللوحة الرئيسية والتقرير السنوي'!F5 / SUM('يناير'!D33:D62, 'فبراير'!D33:D62, 'مارس'!D33:D62, 'أبريل'!D33:D62, 'مايو'!D33:D62, 'يونيو'!D33:D62, 'يوليو'!D33:D62, 'أغسطس'!D33:D62, 'سبتمبر'!D33:D62, 'أكتوبر'!D33:D62, 'نوفمبر'!D33:D62, 'ديسمبر'!D33:D62)), 0))) +
   30 * IFERROR('الأهداف المالية والادخار'!F2, 0)
   , 0)
```

## 15. شريط SPARKLINE تقدّم الهدف - مع تلوين تدريجي

**الوصف:** يرسم شريطاً أفقياً بنسبة الإنجاز للهدف، ويلوّنه ديناميكياً بحسب التدرّج: أحمر < 33%، كهرماني 33-66%، أخضر > 66%.

**الخليّة المستهدفة:** `V31:X31` (مدموجة) للهدف الأوّل، `V34:X34` للثاني، `V37:X37` للثالث.

### 15.1 الهدف 1 - صندوق الطوارئ

```
=SPARKLINE(
   IFERROR(XLOOKUP("صندوق الطوارئ", 'الأهداف المالية والادخار'!A7:A26, 'الأهداف المالية والادخار'!D7:D26), 0),
   {"charttype","bar"; "max",1; "color1", IF(IFERROR(XLOOKUP("صندوق الطوارئ", 'الأهداف المالية والادخار'!A7:A26, 'الأهداف المالية والادخار'!D7:D26), 0) < 0.33, "#DC2626", IF(IFERROR(XLOOKUP("صندوق الطوارئ", 'الأهداف المالية والادخار'!A7:A26, 'الأهداف المالية والادخار'!D7:D26), 0) < 0.66, "#F59E0B", "#10B981")); "empty","zero"}
)
```

### 15.2 الهدف 2 - شراء منزل

نفس الصيغة بإحلال `"صندوق الطوارئ"` بـ `"شراء منزل"` في كل المواضع الستّة.

### 15.3 الهدف 3 - صندوق التقاعد

نفس الصيغة بإحلال `"صندوق الطوارئ"` بـ `"صندوق التقاعد"`. **ملاحظة:** يتطلّب إضافة صفّ `صندوق التقاعد` يدوياً إلى ورقة الأهداف قبل أن يُرجِع SPARKLINE قيمة ذات معنى.

## 16. نسبة تقدّم الهدف كنصّ منسَّق

**الوصف:** يُلبس النسبة العشريّة (0..1) بتنسيق `"0.0%"` لعرضها بجانب شريط SPARKLINE.

**الخليّة المستهدفة:** `U31` للهدف 1، `U34` للثاني، `U37` للثالث.

```
=TEXT(IFERROR(XLOOKUP("صندوق الطوارئ", 'الأهداف المالية والادخار'!A7:A26, 'الأهداف المالية والادخار'!D7:D26), 0), "0.0%")
```

## 17. التنسيق الشرطي - تلوين نصّ النسبة

**الوصف:** قواعد `Custom formula is` للنطاق `U31, U34, U37`. تستخرج الجزء العددي من النسبة المنسَّقة وتقارنه بالعتبات.

```
=VALUE(SUBSTITUTE($U31,"%","")) < 33      (نصّ #DC2626)
=VALUE(SUBSTITUTE($U31,"%","")) < 66      (نصّ #F59E0B)
=VALUE(SUBSTITUTE($U31,"%","")) >= 66     (نصّ #10B981)
```

## 18. سجلّ \"أحدث المعاملات\" - Variant A عبر المحرّك

**الوصف:** صيغة QUERY واحدة على مصفوفة موحَّدة (588 صفّاً) في `_DashboardEngine!Q2:W`.

**الخليّة المستهدفة:** `H48` في ورقة اللوحة.

```
=QUERY(_DashboardEngine!Q2:W, "select * where Col2 is not null order by Col2 desc limit 5", 0)
```

### 18.1 صيغ بناء المصفوفة (مكرَّرة 12 مرّة)

لكل ورقة شهريّة (مثلاً `يناير`)، اكتب البلوك التالي على المحرّك:

**كتلة الدخل (تبدأ من الصف 2 لـ يناير):**

```
Q2 = =ARRAYFORMULA(IF('يناير'!A10:A28="", "", "يناير"))
R2 = =ARRAYFORMULA('يناير'!A10:A28)
S2 = =ARRAYFORMULA(IF('يناير'!A10:A28="", "", "دخل"))
T2 = =ARRAYFORMULA('يناير'!B10:B28)
U2 = =ARRAYFORMULA('يناير'!C10:C28)
V2 = =ARRAYFORMULA('يناير'!E10:E28)
W2 = =ARRAYFORMULA('يناير'!G10:G28)
```

**كتلة المصاريف (تبدأ من الصف 21 لـ يناير، بعد 19 صفّ دخل):**

```
Q21 = =ARRAYFORMULA(IF('يناير'!A33:A62="", "", "يناير"))
R21 = =ARRAYFORMULA('يناير'!A33:A62)
S21 = =ARRAYFORMULA(IF('يناير'!A33:A62="", "", "مصروف"))
T21 = =ARRAYFORMULA('يناير'!B33:B62)
U21 = =ARRAYFORMULA('يناير'!C33:C62)
V21 = =ARRAYFORMULA('يناير'!E33:E62)
W21 = =ARRAYFORMULA('يناير'!G33:G62)
```

كرّر للأشهر الـ 11 المتبقية، صفّاً تلو الآخر، عبر إحلال اسم الورقة.

> إجمالي الصفوف المساعدة: 12 × (19 + 30) = 588 صفّاً، يبدأ شهر فبراير من الصف 51، مارس من الصف 100، إلخ.

## 19. سجلّ \"أحدث المعاملات\" - Variant B بصيغة Array Literal

**الوصف:** صيغة QUERY واحدة على Array Literal دون محرّك مساعد. الفاصل بين الأعمدة `,` في en-US و`\` في إعدادات اللغات الأخرى. الفاصل بين الصفوف `;` في كل الـ Locales.

**الخليّة المستهدفة:** `H48`.

```
=QUERY(
   {
     ARRAYFORMULA(IF('يناير'!A10:A28="", "", "يناير")) , 'يناير'!A10:A28 , ARRAYFORMULA(IF('يناير'!A10:A28="", "", "دخل"))    , 'يناير'!B10:B28 , 'يناير'!C10:C28 , 'يناير'!E10:E28 , 'يناير'!G10:G28
     ;
     ARRAYFORMULA(IF('يناير'!A33:A62="", "", "يناير")) , 'يناير'!A33:A62 , ARRAYFORMULA(IF('يناير'!A33:A62="", "", "مصروف")) , 'يناير'!B33:B62 , 'يناير'!C33:C62 , 'يناير'!E33:E62 , 'يناير'!G33:G62
     ;
     ARRAYFORMULA(IF('فبراير'!A10:A28="", "", "فبراير")) , 'فبراير'!A10:A28 , ARRAYFORMULA(IF('فبراير'!A10:A28="", "", "دخل"))    , 'فبراير'!B10:B28 , 'فبراير'!C10:C28 , 'فبراير'!E10:E28 , 'فبراير'!G10:G28
     ;
     ARRAYFORMULA(IF('فبراير'!A33:A62="", "", "فبراير")) , 'فبراير'!A33:A62 , ARRAYFORMULA(IF('فبراير'!A33:A62="", "", "مصروف")) , 'فبراير'!B33:B62 , 'فبراير'!C33:C62 , 'فبراير'!E33:E62 , 'فبراير'!G33:G62
     ;
     /* ... كرّر نفس الكتلتين للأشهر العشرة المتبقية ... */
   },
   "select * where Col2 is not null order by Col2 desc limit 5",
   0
)
```

## 22. مُغلِّف `TEXT` لتنسيق العرض بسلسلة `rng_ActiveFormat`

**الوصف:** يُلبس قيمة عددية بسلسلة التنسيق المالي للعملة الرئيسيّة النشطة. **نتيجته نصّ**، فلا يصلح للحسابات الإضافيّة.

**أمثلة استعمال في اللوحة:**

```
=TEXT(B5, rng_ActiveFormat)        (إجمالي الدخل كنصّ منسَّق)
=TEXT(J5, rng_ActiveFormat)        (صافي الربح كنصّ منسَّق)
=TEXT(M48, rng_ActiveFormat)       (مبلغ المعاملة الأخيرة كنصّ منسَّق)
```

> الخيار الأفضل عمليّاً هو تنسيق العمود بأكمله عبر `Format → Number → Custom number format` بسلسلة `rng_ActiveFormat`، فيظلّ المحتوى عددياً قابلاً للحسابات.
