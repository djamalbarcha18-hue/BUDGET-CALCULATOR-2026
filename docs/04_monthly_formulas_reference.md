# المرجع السريع لصيغ الأوراق الشهرية
## نظام مالي ذكي متكامل - طبقة الـ 12 شهراً

> هذا الملف يجمع **كل الصيغ** المعتمدة في وثيقة العمارة [`03_monthly_sheets_architecture.md`](./03_monthly_sheets_architecture.md) في كتل قابلة للنسخ المباشر إلى أيّ خلية في Google Sheets. لكل صيغة وصف عربي قصير، والمرجع المرجعي لخلية الكتابة الموصى بها، ومتغيّرات بديلة (per-row / `ARRAYFORMULA` / `IFS` / `INDEX-MATCH` / تنسيق `TEXT`).
>
> كل الصيغ تستهلك حصراً النطاقات المُسمّاة `rng_*` المُعرَّفة في [وثيقة المرحلة الأولى](./01_settings_and_exchange_rates.md#4-النطاقات-المسمّاة-named-ranges-configuration).

---

## 1. صيغة الفرق في كتلة الدخل (Per-row)

**الوصف:** الفرق = `الدخل الفعلي − الدخل المتوقع`. موجبة = تجاوز الهدف. سالبة = عجز.

**الخلية المستهدفة:** `F10` (تُسحب يدوياً حتى `F28`).

```
=IFERROR(IF(OR(D10="", E10=""), "", E10-D10), "")
```

---

## 2. صيغة الفرق في كتلة الدخل (ARRAYFORMULA لكامل العمود)

**الوصف:** بديل أنظف يُكتب مرّة واحدة في `F10` ويُغطّي النطاق `F10:F28` تلقائياً.

```
=ARRAYFORMULA(IF((D10:D28="")+(E10:E28="")>0, "", E10:E28 - D10:D28))
```

---

## 3. صيغة الفرق في كتلة المصاريف (Per-row)

**الوصف:** الفرق = `المصروف المتوقع − المصروف الفعلي`. موجبة = توفير. سالبة = تجاوز ميزانية.

**الخلية المستهدفة:** `F33` (تُسحب يدوياً حتى `F62`).

```
=IFERROR(IF(OR(D33="", E33=""), "", D33-E33), "")
```

---

## 4. صيغة الفرق في كتلة المصاريف (ARRAYFORMULA لكامل العمود)

**الوصف:** بديل ARRAYFORMULA يُكتب مرّة واحدة في `F33` فيغطّي `F33:F62`.

```
=ARRAYFORMULA(IF((D33:D62="")+(E33:E62="")>0, "", D33:D62 - E33:E62))
```

---

## 5. التنبيه على مستوى الصف (Per-row Alert) - نسخة `IF` المتداخلة

**الوصف:** يُعيد إحدى ثلاث إشارات (🔴 / 🟡 / 🟢) أو خلية فارغة إذا كان الصف ناقصاً. الترتيب الإلزامي: حارس الفراغ أوّلاً، ثمّ الأحمر، ثمّ الأصفر، ثمّ الأخضر افتراضياً.

**الخلية المستهدفة:** `H33` (تُسحب يدوياً حتى `H62`).

```
=IF(OR(D33="", E33=""), "",
   IF(E33 > D33, "🔴 تجاوز الميزانية",
      IF(E33 >= 0.9 * D33, "🟡 اقتراب من الحد",
         "🟢 أداء مالي ممتاز")))
```

---

## 6. التنبيه على مستوى الصف - نسخة `IFS` المكافئة

**الوصف:** نفس المنطق بصيغة `IFS` الأنظف.

```
=IFS(
   OR(D33="", E33=""), "",
   E33 > D33,             "🔴 تجاوز الميزانية",
   E33 >= 0.9 * D33,      "🟡 اقتراب من الحد",
   TRUE,                   "🟢 أداء مالي ممتاز"
)
```

---

## 7. الإجماليات الأساسية للوحة المؤشّرات

**الوصف:** أربعة إجماليات تُغذّي كل المؤشّرات الأخرى. يجب تنسيق الخلايا بسلسلة `rng_ActiveFormat`.

```
B3 ← =SUM(D10:D28)        (إجمالي الدخل المتوقع)
B4 ← =SUM(E10:E28)        (إجمالي الدخل الفعلي)
D3 ← =SUM(D33:D62)        (إجمالي المصروف المتوقع)
D4 ← =SUM(E33:E62)        (إجمالي المصروف الفعلي)
```

---

## 8. صافي الفائض / العجز

**الوصف:** الدخل الفعلي ناقص المصروف الفعلي. القيمة الموجبة = فائض.

**الخلية المستهدفة:** `B5`.

```
=B4-D4
```

---

## 9. نسبة الادخار

**الوصف:** `(الدخل الفعلي − المصروف الفعلي) ÷ الدخل الفعلي`. تُنسَّق كنسبة مئويّة. `IFERROR` تحمي من القسمة على صفر.

**الخلية المستهدفة:** `F3`.

```
=IFERROR((B4-D4)/B4, 0)
```

---

## 10. نسبة الإنفاق

**الوصف:** `المصروف الفعلي ÷ الدخل الفعلي`. تُنسَّق كنسبة مئويّة.

**الخلية المستهدفة:** `F4`.

```
=IFERROR(D4/B4, 0)
```

---

## 11. أعلى فئة استنزاف - نسخة `XLOOKUP` (الموصى بها)

**الوصف:** يُجمِّع `SUMIF` المصروف الفعلي لكل فئة من `rng_ExpenseCategories`، ثم يأخذ `MAX`، ثم يسترجع اسم الفئة الموافق عبر `XLOOKUP`. ARRAYFORMULA حول SUMIF تضمن إنتاج مصفوفة بحجم عدد الفئات.

**الخلية المستهدفة:** `D5`.

```
=IFERROR(
   XLOOKUP(
      MAX(ARRAYFORMULA(SUMIF(B33:B62, rng_ExpenseCategories, E33:E62))),
      ARRAYFORMULA(SUMIF(B33:B62, rng_ExpenseCategories, E33:E62)),
      rng_ExpenseCategories
   ),
   ""
)
```

---

## 12. أعلى فئة استنزاف - نسخة `INDEX/MATCH` (للتوافق الواسع)

**الوصف:** نفس المنطق بدون `XLOOKUP`، يصلح للمصنّفات على إصدارات قديمة أو لمن يفضّل `INDEX/MATCH`.

**الخلية المستهدفة:** `D5` (بديل عن الفقرة 11).

```
=IFERROR(
   INDEX(
      rng_ExpenseCategories,
      MATCH(
         MAX(ARRAYFORMULA(SUMIF(B33:B62, rng_ExpenseCategories, E33:E62))),
         ARRAYFORMULA(SUMIF(B33:B62, rng_ExpenseCategories, E33:E62)),
         0
      )
   ),
   ""
)
```

---

## 13. مبلغ أعلى فئة استنزاف

**الوصف:** القيمة العظمى لمجاميع الفئات. تُنسَّق بسلسلة `rng_ActiveFormat`.

**الخلية المستهدفة:** `F5`.

```
=IFERROR(MAX(ARRAYFORMULA(SUMIF(B33:B62, rng_ExpenseCategories, E33:E62))), 0)
```

---

## 14. مؤشّر التنبيه الذكي على مستوى الورقة - نسخة `IF`

**الوصف:** يطبّق نفس عتبات التنبيه على إجمالي المصروف الفعلي (`D4`) مقابل إجمالي المصروف المتوقع (`D3`). الشرط `D3=0` يحمي من تنبيه أحمر كاذب قبل إدخال أيّ ميزانية.

**الخلية المستهدفة:** `F2`.

```
=IF(OR(D3="", D4="", D3=0), "",
   IF(D4 > D3, "🔴 تجاوز الميزانية",
      IF(D4 >= 0.9 * D3, "🟡 اقتراب من الحد",
         "🟢 أداء مالي ممتاز")))
```

---

## 15. مؤشّر التنبيه الذكي على مستوى الورقة - نسخة `IFS`

**الوصف:** نسخة `IFS` المكافئة.

**الخلية المستهدفة:** `F2` (بديل عن الفقرة 14).

```
=IFS(
   OR(D3="", D4="", D3=0), "",
   D4 > D3,                 "🔴 تجاوز الميزانية",
   D4 >= 0.9 * D3,          "🟡 اقتراب من الحد",
   TRUE,                    "🟢 أداء مالي ممتاز"
)
```

```

D29 ← =SUM(D10:D28)           (إجمالي الدخل المتوقع)
E29 ← =SUM(E10:E28)           (إجمالي الدخل الفعلي)
F29 ← =E29-D29                (إجمالي الفرق)
```

## 17. إجماليات صف الذيل (Totals Row) لكتلة المصاريف

**الوصف:** صيغ الصف 63 لكتلة المصاريف.

```
A63 ← الإجمالي                (تسمية)
D63 ← =SUM(D33:D62)           (إجمالي المصروف المتوقع)
E63 ← =SUM(E33:E62)           (إجمالي المصروف الفعلي)
F63 ← =D63-E63                (إجمالي الفرق - موجب يعني توفير)
```

## 18. مُغلِّف `TEXT` لتنسيق العرض بسلسلة `rng_ActiveFormat`

**الوصف:** يُلبس قيمة عددية بسلسلة التنسيق المالي للعملة الرئيسية النشطة. **نتيجته نصّ**، فلا يصلح للحسابات الإضافية. يُستخدم في خلايا العرض النهائية (لوحات المعلومات والتقارير).

**أمثلة استعمال:**

```
=TEXT(B5, rng_ActiveFormat)        (صافي الفائض كنصّ منسَّق)
=TEXT(F5, rng_ActiveFormat)        (مبلغ أعلى فئة استنزاف كنصّ منسَّق)
=TEXT(SUM(E33:E62), rng_ActiveFormat)
```

## 19. التحقّق المنسدل (Data Validation) للأعمدة المرجعيّة

**الوصف:** هذه ليست صيغاً خلويّة بل قيم تُلصَق في حقل مصدر `Dropdown (from a range)` في نافذة `Data validation`.

```
B10:B28        ←  =rng_IncomeCategories      (فئة الدخل)
B33:B62        ←  =rng_ExpenseCategories     (فئة المصاريف)
G10:G28        ←  =rng_PaymentMethods        (طريقة الدفع - دخل)
G33:G62        ←  =rng_PaymentMethods        (طريقة الدفع - مصاريف)
```

## 20. صيغ التنسيق الشرطي (Conditional Formatting Custom Formulas)

**الوصف:** هذه ليست صيغاً خلويّة بل قيم تُلصَق في حقل `Custom formula is` بنافذة `Format → Conditional formatting`. تُطبَّق على النطاق `H33:H62`.

```
=$H10="🔴 تجاوز الميزانية"      (خلفية حمراء)
=$H10="🟡 اقتراب من الحد"        (خلفية كهرمانيّة)
=$H10="🟢 أداء مالي ممتاز"       (خلفية خضراء)
```

## 21. مُجمِّع شهري لاستهلاكه في ورقة "الملخّص السنوي" (لاحقاً)

**الوصف:** نمط `SUMIF` يُجمِّع المصروف الفعلي لفئة واحدة في **ورقة شهر محدّد بالاسم**. يُستخدم لاحقاً في بناء ورقة "الملخّص السنوي" التي تتجمّع فيها 12 ورقة في جدول واحد.

### 21.أ تجميع فئة بعينها في شهر بعينه (مرجع مباشر)

**مثال: مجموع المصروف الفعلي لفئة `الطعام` في ورقة `يناير`:**

```
=SUMIF('يناير'!B33:B62, "الطعام", 'يناير'!E33:E62)
```

### 21.ب نسخة جدوليّة موجَّهة بالنطاق المُسمّى

**الوصف:** اكتب الصيغة في خليّة من ورقة الملخّص السنوي حيث:
- `$A2` يحتوي اسم الفئة المسحوب من `rng_ExpenseCategories`.
- `B$1` يحتوي اسم الشهر (مثلاً `يناير`).

```
=SUMIF(INDIRECT("'"&B$1&"'!B33:B62"), $A2, INDIRECT("'"&B$1&"'!E33:E62"))
```

> سحب هذه الصيغة أفقياً عبر 12 شهراً وعمودياً عبر 12 فئة يُنتج جدول الملخّص السنوي بالكامل.

### 21.ج تجميع كامل للمصروف الفعلي لكل شهر (إجمالي شهر)

```
=SUM(INDIRECT("'"&B$1&"'!E33:E62"))
```

### 21.د تجميع كامل لإجمالي الدخل الفعلي لكل شهر

```
=SUM(INDIRECT("'"&B$1&"'!E10:E28"))
```

### 21.هـ صافي شهر بعينه

```
=SUM(INDIRECT("'"&B$1&"'!E10:E28")) - SUM(INDIRECT("'"&B$1&"'!E33:E62"))
```

## 22. سحب اسم الشهر من اسم الورقة (اختياري)

**الوصف:** صيغة لمن يرغب بأن تكون خليّة `D2` (الشهر) ديناميكيّة بدل كتابتها يدوياً. تتطلّب أن يكون المصنّف محفوظاً ومُسمّى.

```
=MID(CELL("filename", A1), FIND("]", CELL("filename", A1))+1, 100)
```

> لو لم يكن المصنّف محفوظاً، تُرجِع الصيغة سلسلة فارغة. الإدخال اليدوي لاسم الشهر هو الأسلوب الموصى به للأوراق الشهرية لأنّه يدعم الإحالات المتقاطعة في ورقة "الملخّص السنوي" بأمان أعلى.
