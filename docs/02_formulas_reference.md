# المرجع السريع للصيغ
## نظام مالي ذكي متكامل - طبقة "الإعدادات وأسعار الصرف"

> هذا الملف يجمع كل الصيغ المعتمدة في وثيقة العمارة [`01_settings_and_exchange_rates.md`](./01_settings_and_exchange_rates.md) في كتل قابلة للنسخ المباشر إلى أيّ خلية في Google Sheets. لكل صيغة وصف عربي قصير وصيغة مكافئة بالنطاقات المُسمّاة.

---

## 1. استرجاع سعر الصرف لعملة محددة

**الوصف:** يبحث عن رمز العملة في جدول العملات ويعيد سعر صرفها مقابل الدولار الأمريكي (وحدة العملة لكل 1 USD).

**نسخة المراجع المباشرة:**
```
=XLOOKUP("SAR", 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$C$7:$C$20)
```

**نسخة النطاقات المُسمّاة (المُعتمدة):**
```
=XLOOKUP("SAR", rng_Currencies, rng_CurrencyRates)
```

---

## 2. استرجاع اسم العملة بالعربية من الرمز

**الوصف:** يحوّل الرمز (مثلاً `AED`) إلى الاسم الكامل بالعربية (`الدرهم الإماراتي`) لأغراض العرض والتقارير.

**نسخة المراجع المباشرة:**
```
=XLOOKUP(B3, 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$B$7:$B$20)
```

**نسخة النطاقات المُسمّاة:**
```
=XLOOKUP(rng_MainCurrency, rng_Currencies, rng_CurrencyNames)
```

---

## 3. استرجاع سلسلة التنسيق المالي للعملة الرئيسية

**الوصف:** يعيد سلسلة التنسيق الرقمي الموافقة للعملة الرئيسية المختارة (مثلاً `#,##0.00 "ر.س"`) لتُستخدم كمصدر مرجعي وحيد للتنسيق.

**نسخة المراجع المباشرة:**
```
=XLOOKUP(B3, 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$D$7:$D$20)
```

**نسخة النطاقات المُسمّاة (تُكتب في `B4` من ورقة الإعدادات):**
```
=XLOOKUP(rng_MainCurrency, rng_Currencies, rng_FormatStrings)
```

---

## 4. صيغة التحويل الديناميكية لخلية واحدة (USD كعملة جسر)

**الوصف:** تحوّل مبلغاً من عملة المعاملة (`E2`) إلى العملة الرئيسية للعرض، باستخدام الدولار كعملة جسر.

**نسخة المراجع المباشرة (داخل ورقة شهرية):**
```
=D2 * XLOOKUP('الإعدادات وأسعار الصرف'!$B$3, 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$C$7:$C$20) / XLOOKUP(E2, 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$C$7:$C$20)
```

**نسخة النطاقات المُسمّاة (الموصى بها):**
```
=D2 * XLOOKUP(rng_MainCurrency, rng_Currencies, rng_CurrencyRates) / XLOOKUP(E2, rng_Currencies, rng_CurrencyRates)
```

---

## 5. صيغة التحويل بأسلوب `LET` المُحسَّنة

**الوصف:** نفس منطق التحويل أعلاه لكن باستخدام `LET` لتجنب إعادة الحساب وتحسين الأداء وقابلية القراءة.

```
=LET(
   amt,      D2,
   txnCur,   E2,
   rateMain, XLOOKUP(rng_MainCurrency, rng_Currencies, rng_CurrencyRates),
   rateTxn,  XLOOKUP(txnCur,           rng_Currencies, rng_CurrencyRates),
   IFERROR( amt * rateMain / rateTxn, 0 )
)
```

---

## 6. تحويل عمود معاملات بأكمله عبر `ARRAYFORMULA`

**الوصف:** صيغة وحيدة تُكتب في `F2` من الورقة الشهرية لتغطية كامل العمود تلقائياً، وتُرجع نصاً فارغاً للصفوف الخالية.

**نسخة المراجع المباشرة:**
```
=ARRAYFORMULA(
   IF(D2:D="", "",
      D2:D
      * XLOOKUP('الإعدادات وأسعار الصرف'!$B$3, 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$C$7:$C$20)
      / IFERROR( XLOOKUP(E2:E, 'الإعدادات وأسعار الصرف'!$A$7:$A$20, 'الإعدادات وأسعار الصرف'!$C$7:$C$20), 1 )
   )
)
```

**نسخة النطاقات المُسمّاة (المُعتمدة):**
```
=ARRAYFORMULA(
   IF(D2:D="", "",
      D2:D
      * XLOOKUP(rng_MainCurrency, rng_Currencies, rng_CurrencyRates)
      / IFERROR( XLOOKUP(E2:E, rng_Currencies, rng_CurrencyRates), 1 )
   )
)
```

---

## 7. قوائم منسدلة ديناميكية عبر `FILTER`

**الوصف:** تنتج قائمة منسدلة تتجاهل الخلايا الفارغة وتلتقط أيّ توسعة مستقبلية في القائمة المصدر.

**فئات الدخل:**
```
=FILTER(rng_IncomeCategories, rng_IncomeCategories <> "")
```

**فئات المصاريف:**
```
=FILTER(rng_ExpenseCategories, rng_ExpenseCategories <> "")
```

**طرق الدفع:**
```
=FILTER(rng_PaymentMethods, rng_PaymentMethods <> "")
```

---

## 8. قائمة الفئة المشروطة (دخل أو مصروف) في الورقة الشهرية

**الوصف:** عمود مساعد يبدّل بين قائمة الدخل وقائمة المصاريف بحسب نوع المعاملة في العمود `B`.

```
=IF(B2="دخل",     FILTER(rng_IncomeCategories,  rng_IncomeCategories  <> ""),
 IF(B2="مصروف",  FILTER(rng_ExpenseCategories, rng_ExpenseCategories <> ""),
                  ""))
```

---

## 9. عرض المبلغ المحوَّل بسلسلة التنسيق الفعّالة

**الوصف:** يُلبس القيمة العددية بالتنسيق الرقمي للعملة الرئيسية النشطة.

**نسخة المراجع المباشرة:**
```
=TEXT(F2, 'الإعدادات وأسعار الصرف'!$B$4)
```

**نسخة النطاقات المُسمّاة:**
```
=TEXT(F2, rng_ActiveFormat)
```

---

## 10. التحديث الآلي لسعر الصرف عبر `GOOGLEFINANCE` (اختياري)

**الوصف:** بديل لإدخال السعر يدوياً، مع قيمة احتياطية في حال عدم توفّر العملة لدى Google Finance.

```
=IFERROR( GOOGLEFINANCE("CURRENCY:USD"&A7) , 1 )
```

> **تنبيه:** بعض العملات الإقليمية لا يوفّرها `GOOGLEFINANCE` بانتظام؛ يُفضَّل الإبقاء على الإدخال اليدوي للأسعار غير المدعومة.

---

## 11. مجموع المصاريف للشهر بالعملة الرئيسية

**الوصف:** إجمالي عمود "المبلغ بعملة العرض" المُصفّى على نوع "مصروف" - يُستخدم في رأس الورقة الشهرية كملخّص فوري.

```
=SUMIFS(F2:F, B2:B, "مصروف")
```

ولعرضه بالتنسيق المالي للعملة الرئيسية:
```
=TEXT( SUMIFS(F2:F, B2:B, "مصروف") , rng_ActiveFormat )
```

---

## 12. مجموع الدخل للشهر بالعملة الرئيسية

**الوصف:** نظير الصيغة السابقة لكن للدخل.

```
=SUMIFS(F2:F, B2:B, "دخل")
```

---

## 13. صافي الفائض/العجز الشهري

**الوصف:** الفرق بين الدخل والمصاريف بالعملة الرئيسية.

```
=SUMIFS(F2:F, B2:B, "دخل") - SUMIFS(F2:F, B2:B, "مصروف")
```
