# المرجع السريع لصيغ ورقة الأهداف المالية والادخار
## نظام مالي ذكي متكامل - طبقة الأهداف وصندوق الطوارئ

> هذا الملف يجمع **كل الصيغ** المعتمدة في وثيقة العمارة [`05_savings_goals_architecture.md`](./05_savings_goals_architecture.md) في كتل قابلة للنسخ المباشر إلى أيّ خلية في Google Sheets. لكل صيغة وصف عربي قصير، والمرجع المرجعي لخلية الكتابة الموصى بها، ومتغيّرات بديلة (per-row / `ARRAYFORMULA` / `IFS` / `IF` متداخلة / DATEDIF / حسابيّة).
>
> النطاق المُسمّى الوحيد المُستهلَك من المرحلة الأولى هو `rng_ActiveFormat` (لتنسيق المبالغ داخل صيغة التوصية الذكيّة).

---

## 1. نسبة الإنجاز - صيغة الصفّ الواحد (Per-row)

**الوصف:** نسبة الإنجاز = `المبلغ المدخر حالياً ÷ التكلفة التقديريّة`. تُنسَّق كنسبة مئويّة. `IFERROR` تحمي من القسمة على صفر.

**الخلية المستهدفة:** `D7` (تُسحب يدوياً حتى `D26`).

```
=IFERROR(C7/B7, 0)
```

---

## 2. نسبة الإنجاز - بديل `ARRAYFORMULA` لكامل العمود

**الوصف:** يُكتب مرّة واحدة في `D7` فيغطّي النطاق `D7:D26`. حارس الصف الفارغ يمنع ظهور `0%` في الصفوف غير المُدخَلة.

```
=ARRAYFORMULA(IF(B7:B26="", "", IFERROR(C7:C26/B7:B26, 0)))
```

---

## 3. الأشهر المتبقية - نسخة `DATEDIF` (الموصى بها)

**الوصف:** عدد الأشهر الكاملة بين تاريخ اليوم والموعد المستهدف. `MAX(0, ...)` يحمي من القيم السالبة بعد انقضاء الموعد. `IFERROR` يحمي من خطأ `#NUM!` ومن الخلية الفارغة.

**الخلية المستهدفة:** `F7` (تُسحب يدوياً حتى `F26`).

```
=IFERROR(MAX(0, DATEDIF(TODAY(), E7, "M")), 0)
```

---

## 4. الأشهر المتبقية - نسخة حسابيّة صرفة (Arithmetic Variant)

**الوصف:** بديل لمن ترفض إعدادات منطقتهم `DATEDIF` لبعض أزواج التواريخ. يستخدم متوسط طول الشهر الميلادي `30.4375 = 365.25 / 12`.

**الخلية المستهدفة:** `F7` (بديل عن الفقرة 3).

```
=IFERROR(MAX(0, ROUND((E7-TODAY())/30.4375, 0)), 0)
```

---

## 5. الأشهر المتبقية - بديل `ARRAYFORMULA` (نسخة DATEDIF)

**الوصف:** يُكتب مرّة واحدة في `F7` فيغطّي النطاق `F7:F26`. إن لم يعمل `DATEDIF` داخل `ARRAYFORMULA` على إصدار Google Sheets المستخدم، استبدله بالبديل الحسابي في الفقرة 6.

```
=ARRAYFORMULA(IF(E7:E26="", "", IFERROR(MAX(0, DATEDIF(TODAY(), E7:E26, "M")), 0)))
```

---

## 6. الأشهر المتبقية - بديل `ARRAYFORMULA` (نسخة حسابيّة)

**الوصف:** نسخة `ARRAYFORMULA` آمنة لكل الإصدارات.

```
=ARRAYFORMULA(IF(E7:E26="", "", IFERROR(MAX(0, ROUND((E7:E26-TODAY())/30.4375, 0)), 0)))
```

---

## 7. القسط الشهري المطلوب - صيغة الصفّ الواحد (Per-row)

**الوصف:** المبلغ الذي يلزم ادّخاره شهرياً للوصول للهدف في موعده. إذا اكتمل الهدف يُعاد `0`. `IFERROR` تحمي من القسمة على صفر عندما `F7 = 0` (الموعد منتهٍ).

**الخلية المستهدفة:** `G7` (تُسحب يدوياً حتى `G26`). تُنسَّق الخليّة بسلسلة `rng_ActiveFormat`.

```
=IF(C7>=B7, 0, IFERROR((B7-C7)/F7, 0))
```

---

## 8. القسط الشهري المطلوب - بديل `ARRAYFORMULA` لكامل العمود

**الوصف:** يُكتب مرّة واحدة في `G7` فيغطّي النطاق `G7:G26`.

```
=ARRAYFORMULA(IF(B7:B26="", "", IF(C7:C26>=B7:B26, 0, IFERROR((B7:B26 - C7:C26)/F7:F26, 0))))
```

---

## 9. الحالة - نسخة `IFS` (الموصى بها)

**الوصف:** يُعيد إحدى ثلاث إشارات (🟢 / 🟡 / ⚪) أو خلية فارغة إذا كان الصف ناقصاً. الترتيب الإلزامي: حارس الفراغ أوّلاً، ثمّ الإكمال (≥ 100%)، ثمّ بدء الادخار (≥ 1%)، ثمّ الافتراضي (لم يبدأ بعد).

**الخلية المستهدفة:** `H7` (تُسحب يدوياً حتى `H26`).

```
=IFS(
   B7="",                              "",
   IFERROR(C7/B7, 0) >= 1,             "🟢 مكتمل",
   IFERROR(C7/B7, 0) >= 0.01,          "🟡 قيد الادخار",
   TRUE,                                "⚪ لم يبدأ بعد"
)
```

---

## 10. الحالة - نسخة `IF` المتداخلة المكافئة

**الوصف:** نفس المنطق بصيغة `IF` المتداخلة لمن يفضّلها أو لمصنّفات قديمة لا تدعم `IFS`.

```
=IF(B7="", "",
   IF(IFERROR(C7/B7, 0) >= 1, "🟢 مكتمل",
      IF(IFERROR(C7/B7, 0) >= 0.01, "🟡 قيد الادخار",
         "⚪ لم يبدأ بعد")))
```

---

## 11. التوصية الذكيّة - الصيغة الكاملة (`IFS`)

**الوصف:** سلسلة عربيّة ديناميكيّة تُولَّد بـ `IFS` تأخذ في الاعتبار حالة الإكمال، انتهاء الموعد، والقسط الشهري المطلوب مقابل الزمن المتبقّي. المبلغ داخل النصّ يُنسَّق ديناميكيّاً عبر `TEXT(G7, rng_ActiveFormat)`.

**الخلية المستهدفة:** `I7` (تُسحب يدوياً حتى `I26`).

**ترتيب الفروع وكيف يُختار كلٌّ منها:**

| الفرع | الشرط | كيف يُختار |
|---|---|---|
| 1 | `B7=""` | لا يوجد هدف مُعرَّف على هذا الصف، فيُعاد نصّ فارغ. |
| 2 | `IFERROR(C7/B7, 0) >= 1` | الهدف محقَّق (نسبة إنجاز ≥ 100%) - رسالة تهنئة. |
| 3 | `F7 = 0` (وضمنياً نسبة الإنجاز < 100% لأنّ الفرع 2 سبقه) | الموعد المستهدف انقضى أو هو اليوم بالضبط دون اكتمال - رسالة تنبيه. |
| 4 | `F7 > 24` | أفق طويل (> سنتين) - رسالة طمأنة بنبرة هادئة. |
| 5 | `F7 > 6` (وضمنياً ≤ 24 لأنّ الفرع 4 سبقه) | جهد متوازن (نصف سنة إلى سنتين) - رسالة التزام. |
| 6 | `TRUE` (يلتقط `F7 ≤ 6`) | إلحاح (ستّة أشهر أو أقلّ) - رسالة استعجال ومراجعة الأولويّات. |

```
=IFS(
   B7="", "",
   IFERROR(C7/B7, 0) >= 1, "🎉 الهدف محقق بالكامل، يمكن إعادة توجيه القسط لهدف جديد.",
   F7 = 0, "⚠️ الموعد المستهدف انتهى دون اكتمال الهدف، يجب تمديد المدة أو زيادة الادخار.",
   F7 > 24, "⏳ لديك وقت كافٍ، خصص " & TEXT(G7, rng_ActiveFormat) & " شهرياً للوصول للهدف خلال " & F7 & " شهراً.",
   F7 > 6, "🟡 الزمن متوسط، التزم بـ " & TEXT(G7, rng_ActiveFormat) & " شهرياً ولا تؤجّل الادخار.",
   TRUE, "🔴 الموعد قريب جداً، يلزم ادخار " & TEXT(G7, rng_ActiveFormat) & " شهرياً وقد يتطلب الأمر مراجعة الأولويات."
)
```

---

## 12. مُغلِّف `TEXT` لتنسيق المبلغ بسلسلة `rng_ActiveFormat`

**الوصف:** يُلبس قيمة عددية بسلسلة التنسيق المالي للعملة الرئيسية النشطة. **نتيجته نصّ**، فلا يصلح للحسابات الإضافية. هو الطريقة الوحيدة المسموح بها لإدراج رمز عملة داخل سلسلة التوصية الذكيّة.

```
=TEXT(G7, rng_ActiveFormat)
```

**أمثلة استخدام أخرى داخل الورقة:**

```
=TEXT(B2, rng_ActiveFormat)        (عرض إجمالي تكلفة الأهداف كنصّ منسَّق)
=TEXT(D2, rng_ActiveFormat)        (عرض إجمالي المدخر كنصّ منسَّق)
=TEXT(B3, rng_ActiveFormat)        (عرض إجمالي القسط الشهري المطلوب كنصّ منسَّق)
```

---

## 13. لوحة المؤشّرات - إجمالي تكلفة الأهداف

**الوصف:** مجموع التكاليف التقديريّة لكل الأهداف المُدخَلة. تُنسَّق الخلية بسلسلة `rng_ActiveFormat`.

**الخلية المستهدفة:** `B2`.

```
=SUM(B7:B26)
```

---

## 14. لوحة المؤشّرات - إجمالي المدخر

**الوصف:** مجموع المبالغ المدخّرة حالياً لكل الأهداف. تُنسَّق الخلية بسلسلة `rng_ActiveFormat`.

**الخلية المستهدفة:** `D2`.

```
=SUM(C7:C26)
```

---

## 15. لوحة المؤشّرات - نسبة الإنجاز الكليّة

**الوصف:** نسبة الإنجاز الإجماليّة عبر كل الأهداف = `إجمالي المدخر ÷ إجمالي التكلفة`. تُنسَّق كنسبة مئويّة. `IFERROR` تحمي من القسمة على صفر عند جدول فارغ.

**الخلية المستهدفة:** `F2`.

```
=IFERROR(SUM(C7:C26)/SUM(B7:B26), 0)
```

---

## 16. لوحة المؤشّرات - إجمالي القسط الشهري المطلوب

**الوصف:** مجموع الأقساط الشهريّة لكلّ الأهداف ذات الحالة `🟡 قيد الادخار` فقط (الأهداف المكتملة لا تتطلّب قسطاً، والأهداف التي لم تبدأ ستُلتقَط لاحقاً عند بدء الادخار فيها). تُنسَّق الخلية بسلسلة `rng_ActiveFormat`.

**الخلية المستهدفة:** `B3`.

```
=SUMIF(H7:H26, "🟡 قيد الادخار", G7:G26)
```

---

## 17. لوحة المؤشّرات - عدد الأهداف المكتملة

**الوصف:** عدّ الصفوف ذات الحالة `🟢 مكتمل` بالضبط.

**الخلية المستهدفة:** `D3`.

```
=COUNTIF(H7:H26, "🟢 مكتمل")
```

---

## 18. لوحة المؤشّرات - عدد الأهداف قيد الادخار

**الوصف:** عدّ الصفوف ذات الحالة `🟡 قيد الادخار` بالضبط.

**الخلية المستهدفة:** `F3`.

```
=COUNTIF(H7:H26, "🟡 قيد الادخار")
```

---

## 19. لوحة المؤشّرات - عدد الأهداف التي لم تبدأ

**الوصف:** عدّ الصفوف ذات الحالة `⚪ لم يبدأ بعد` بالضبط.

**الخلية المستهدفة:** `B4`.

```
=COUNTIF(H7:H26, "⚪ لم يبدأ بعد")
```

---

## 20. صيغ التنسيق الشرطي (Conditional Formatting Custom Formulas)

**الوصف:** هذه ليست صيغاً خلويّة بل قيم تُلصَق في حقل `Custom formula is` بنافذة `Format → Conditional formatting`. تُطبَّق على النطاق المذكور أمام كل صيغة.

**على النطاق `H7:H26` (عمود الحالة):**

```
=$H7="🟢 مكتمل"           (خلفية خضراء)
=$H7="🟡 قيد الادخار"      (خلفية كهرمانيّة)
=$H7="⚪ لم يبدأ بعد"      (خلفية رماديّة فاتحة)
```

**على النطاق `D7:D26` (عمود نسبة الإنجاز) - قاعدة موصى بها:**

```
=$D7>=1                    (خلفية خضراء فاتحة لإبراز الأهداف المكتملة)
```

---

## 21. مرجع للأوراق المستقبليّة (Cross-Sheet Lookups)

**الوصف:** هذه ليست صيغاً تُكتب في ورقة الأهداف، بل تُستخدَم لاحقاً في ورقة "الملخّص السنوي" (المرحلة القادمة) لقراءة قيم لوحة المؤشّرات الكليّة من ورقة الأهداف.

```
='الأهداف المالية والادخار'!B2        (إجمالي تكلفة الأهداف)
='الأهداف المالية والادخار'!D2        (إجمالي المدخر)
='الأهداف المالية والادخار'!F2        (نسبة الإنجاز الكليّة)
='الأهداف المالية والادخار'!B3        (إجمالي القسط الشهري المطلوب)
='الأهداف المالية والادخار'!D3        (عدد الأهداف المكتملة)
='الأهداف المالية والادخار'!F3        (عدد الأهداف قيد الادخار)
='الأهداف المالية والادخار'!B4        (عدد الأهداف التي لم تبدأ)
```
