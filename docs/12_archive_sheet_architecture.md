# 12 – أرشيف السنوات المالية (Financial Years Archive)

> **المرحلة:** Phase 7  
> **الإصدار:** 1.2.0  
> **التاريخ:** يونيو 2026  
> **المطوّر:** Boulahdid Djamal Eddine

---

## 1. نظرة عامة

ورقة أرشيفية شاملة تسحب البيانات **تلقائياً** من سجل المعاملات المركزي
(`_DashboardEngine!Q:W`) باستخدام دوال `SUMIFS` و `COUNTIFS`، وتعرض ملخصاً
سنوياً/شهرياً لكل سنة مالية من 2025 إلى 2035.

## 2. الموقع داخل المصنّف

|--------|--------|--------|

| وصف توضيحي | أرشيف السنوات المالية | `A2:H2` |

---

## 3. هيكل الأعمدة

| العمود | الحقل | النوع | التنسيق |
|--------|-------|-------|---------|
| A | السنة | رقم ثابت | `0` (بدون فواصل) |
| B | الشهر | نص ثابت | يناير → ديسمبر |
| C | إجمالي الدخل | **معادلة SUMIFS** | `#,##0.00` |
| D | إجمالي المصاريف | **معادلة SUMIFS** | `#,##0.00` |
| E | صافي الربح | **معادلة** `=C-D` | `#,##0.00` |
| F | نسبة الادخار % | **معادلة** `=(C-D)/C` | `0.00%` |
| G | معاملات الدخل (عدد) | **معادلة COUNTIFS** | `0` |
| H | معاملات المصاريف (عدد) | **معادلة COUNTIFS** | `0` |

---

## 4. المعادلات التفصيلية

### 4.1 إجمالي الدخل (العمود C)

```
=IFERROR(SUMIFS('_DashboardEngine'!V:V, '_DashboardEngine'!Q:Q, B4, '_DashboardEngine'!S:S, "دخل"), 0)
```

**الشروط:**
- `Q:Q` = اسم الشهر (مطابقة مع العمود B في الأرشيف)
- `S:S` = "دخل" (نوع المعاملة)
- `V:V` = المبلغ (القيمة المُجمَّعة)

### 4.2 إجمالي المصاريف (العمود D)

```
=IFERROR(SUMIFS('_DashboardEngine'!V:V, '_DashboardEngine'!Q:Q, B4, '_DashboardEngine'!S:S, "مصروف"), 0)
```

### 4.3 صافي الربح (العمود E)

```
=C4-D4
```

### 4.4 نسبة الادخار (العمود F)

```
=IFERROR((C4-D4)/C4, 0)
```

### 4.5 عدد معاملات الدخل (العمود G)

```
=IFERROR(COUNTIFS('_DashboardEngine'!Q:Q, B4, '_DashboardEngine'!S:S, "دخل", '_DashboardEngine'!V:V, "<>"), 0)
```

### 4.6 عدد معاملات المصاريف (العمود H)

```
=IFERROR(COUNTIFS('_DashboardEngine'!Q:Q, B4, '_DashboardEngine'!S:S, "مصروف", '_DashboardEngine'!V:V, "<>"), 0)
```

## 5. مصدر البيانات

الأرشيف يقرأ من **سجل المعاملات المتراكم** في `_DashboardEngine!Q:W`:

| العمود | المحتوى |
|--------|---------|
| Q | الشهر (اسم الشهر) |
| R | التاريخ |
| S | النوع ("دخل" أو "مصروف") |
| T | الفئة |
| U | الوصف |
| V | المبلغ |
| W | طريقة الدفع |

يحتوي المحرك على بلوكات ARRAYFORMULA تسحب من الأوراق الشهرية:
- الدخل: 12 شهر × 19 صف = 228 صف
- المصاريف: 12 شهر × 30 صف = 360 صف
- الإجمالي: 588 صف كحد أقصى

## 6. التنسيق الشرطي

### صافي الربح (العمود E)
| الشرط | اللون |
|-------|-------|
| > 0 | `#10B981` (أخضر) |
| < 0 | `#DC2626` (أحمر) |

### نسبة الادخار (العمود F)
| الشرط | اللون |
|-------|-------|
| >= 20% | `#10B981` (أخضر) |
| 1% - 19% | `#F59E0B` (ذهبي) |
| <= 0% | `#DC2626` (أحمر) |

## 7. لوحة الملخص الإجمالي (صف 136-138)

| الخلية | المعادلة | الوصف |
|--------|----------|-------|
| A138 | `=SUM(C4:C135)` | إجمالي الدخل الكلي |
| C138 | `=SUM(D4:D135)` | إجمالي المصاريف الكلية |
| E138 | `=SUM(E4:E135)` | صافي الربح الكلي |
| G138 | `=SUM(G4:G135)+SUM(H4:H135)` | إجمالي المعاملات |

## 8. التصميم والمعايير

- **Dark Mode (Fintech):** خلفية `#0F172A` / `#1F2937` بتلوين متبادل (Zebra)
- **اللغة:** عربية بالكامل لجميع التسميات
- **الأرقام:** لاتينية حصراً — مفروضة عبر `setNumberFormat`
- **الفواصل العشرية:** `#,##0.00` للمبالغ، `0.00%` للنسب
- **RTL:** الورقة مضبوطة من اليمين لليسار
- **الحماية:** الورقة محمية بالكامل (Warning only)
