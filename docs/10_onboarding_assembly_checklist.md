# قائمة المراجعة لتركيب ورقة الترحيب
## نظام مالي ذكي متكامل - Onboarding Assembly Checklist

> هذا الملف يجمع كل خطوات تركيب ورقة `📖 دليل الاستخدام والترحيب` (المرحلة 5) في قائمة قابلة للتنفيذ المباشر، بالإضافة إلى صيغ الـ Hyperlink الجاهزة للنسخ. للتفاصيل المعماريّة وراء كل قرار، راجع [`docs/09_onboarding_welcome_architecture.md`](./09_onboarding_welcome_architecture.md).

## 1. القائمة المختصرة (Quick Checklist)

- [ ] إنشاء ورقة جديدة باسم `📖 دليل الاستخدام والترحيب` (بما فيه الإيموجي).
- [ ] سحب الورقة إلى الموضع رقم 1 (أوّل تبويب).
- [ ] تفعيل `View → Right-to-Left`.
- [ ] تلوين الخلفيّة الكلّية `#0F172A` ولون النصّ `#F1F5F9`.
- [ ] دمج `B2:O4` وكتابة الترويسة المزدوجة.
- [ ] دمج `B6:O9` ولصق النبذة التعريفيّة.
- [ ] بناء البطاقات الثلاث في `B11:F23`، `G11:K23`، `L11:P23`.
- [ ] إضافة الحدود العلويّة الملوَّنة (سماوي / أخضر / برتقالي).
- [ ] ربط Hyperlinks للأوراق المعنيّة (الإعدادات، يناير، اللوحة الرئيسيّة).
- [ ] دمج وكتابة كتلة التوقيع في `B26:O29`.
- [ ] حماية كتلة التوقيع `Data → Protect range → Show a warning`.
- [ ] لصق Footer في الصفوف 32..34.
- [ ] تطبيق طبقة الحماية الناعمة الموسَّعة على باقي الأوراق (انظر القسم 8 من docs/09).
- [ ] التحقّق النهائي عبر `Ctrl+F` على كلمة `Boulahdid` ← يجب أن تُعيد نتيجة واحدة فقط.

## 2. صيغ الـ Hyperlink الجاهزة للنسخ

> استبدِل `<gid>` بالمعرّف الرقمي للورقة الهدف (تجده في URL المتصفّح بعد `#gid=` عند فتح الورقة).

### 2.1 الرابط من البطاقة 1 إلى ورقة الإعدادات

```
=HYPERLINK("#gid=<gid_of_settings_sheet>", "📘 افتح ورقة الإعدادات وأسعار الصرف")
```

**الخليّة المستهدفة:** `B23:F23` (مدموجة).

### 2.2 الرابط من البطاقة 2 إلى ورقة يناير

```
=HYPERLINK("#gid=<gid_of_january_sheet>", "📅 افتح ورقة يناير")
```

**الخليّة المستهدفة:** `G23:K23` (مدموجة).

### 2.3 الرابط من البطاقة 3 إلى اللوحة الرئيسيّة

```
=HYPERLINK("#gid=<gid_of_dashboard_sheet>", "📊 افتح اللوحة الرئيسيّة")
```

**الخليّة المستهدفة:** `L23:P23` (مدموجة).

### 2.4 البديل بدون gid (الأسهل عمليّاً)

استخدم القائمة `Insert → Link → Sheets in this spreadsheet → اختر الورقة المطلوبة`. Google Sheets يُنشئ الـ hyperlink آلياً ويعتني بمعرّف الـ gid في الخلفيّة.

## 3. مراجع نصوص قابلة للصق المباشر

### 3.1 الترويسة (الصف 2..4)

**السطر العربي:**

```

```

**السطر الإنجليزي (تحته بعد Alt+Enter):**

```
BUDGET CALCULATOR 2026
```

### 3.2 النبذة التعريفيّة (`B6:O9`)

```
نظام ميزان المالي الذكي قالب فنتك عربي احترافي مبني على Google Sheets، يجمع محرّك عملات متعدّد، 12 ورقة شهريّة بمحرّك تنبيهات ذكي، نظام أهداف وادّخار مع توصيات آليّة، ولوحة معلومات فاخرة بوضع داكن وتحليلات سنويّة. كل المكوّنات تتدفّق من اليمين إلى اليسار (RTL) وبمصطلحات ماليّة عربيّة احترافيّة.
```

## 4. رسائل التحذير العربيّة الموحَّدة

### 4.1 رسالة لكتل ورقة الترحيب (الترويسة + التوقيع)

```
هذه الكتلة جزء من الهويّة البصريّة للقالب. التعديل غير مرغوب.
```

### 4.2 رسالة لورقة الإعدادات

```
هذه الورقة هي مصدر الحقيقة الوحيد للإعدادات. التعديل قد يكسر كل المصنّف.
```

### 4.3 رسالة للأعمدة المحسوبة (الأهداف + الأوراق الشهريّة + اللوحة الرئيسيّة)

```
هذه الخلية محسوبة آلياً، لا تُعدَّل يدوياً.
```

### 4.4 رسالة لورقة المحرّك الخلفي

```
هذه الورقة جزء من المحرك الخلفي للوحة المعلومات. التعديل قد يكسر الصيغ والرسوم البيانية.
```

| `bg-page` | الخلفيّة الكلّية | `#0F172A` |
